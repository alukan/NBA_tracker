# NBA Schedule Tracker

A simple React Native app built with Expo and TypeScript that lets users follow their favorite NBA teams, view upcoming schedules, and track live game scores.

## Features

### Current Features
- Select favorite NBA teams
- View upcoming games
- Live score tracking
- Game detail screen
- Team-based schedule filtering
- Eastern and Western Conference standings
- Calendar/list schedule views
- Dark mode UI

## Potential Future Features
- Push notifications for games
- Home screen widgets
- Player statistics
- Injury reports
- Team comparison screen
- Predictions and pick'em mode
- Playoff bracket tracker
- Offline caching
- Account syncing
- Personalized game recommendations

## Tech Stack
- React Native
- Expo
- TypeScript
- React Navigation
- TanStack Query
- NativeWind

## APIs
- balldontlie API
- ESPN API (unofficial)

## Goal
The goal of this project is to build a clean and responsive mobile NBA companion app while learning modern React Native development practices.
